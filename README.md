## Instuctions to run:

1.  Clone the project:

```
git clone https://git@github.com:erickbarrosr/professional-portfolio-frontend.git
```

2. After clone open the folder:

```
professional-portfolio
```

3. Open a terminal in the frontend directory and run:

```
ng serve
```
